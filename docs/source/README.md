# Fuente del horario

El archivo `horario.pdf` es la fuente visual proporcionada para el horario inicial.

La página 1 contiene la matriz semanal. La página 2 contiene accesos del sistema académico y no añade bloques al horario.

La transcripción se guardó en `data/schedule.seed.json`. Debido a que la fuente es una captura visual y no una tabla estructurada, el seed está marcado con `reviewRequiredBeforeProduction: true`.
