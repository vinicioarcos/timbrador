# ADR-0003 — Timbrado asistido, no autónomo por defecto

**Estado:** Accepted

## Decisión

El sistema prepara la acción correcta y puede invocar un adaptador autorizado, pero exige confirmación del usuario por defecto.

## Motivo

Separar el recordatorio de la prueba de presencia y evitar que una automatización silenciosa produzca registros que el usuario no haya confirmado.
